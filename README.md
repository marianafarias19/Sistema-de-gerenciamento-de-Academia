# Sistema de Gerenciamento de Academia

Projeto desenvolvido para praticar os fundamentos de Python através da criação de um sistema de cadastro de alunos para uma academia.

## Objetivo

Meu objetivo com este projeto foi sair dos exercícios simples e construir um sistema mais próximo de uma aplicação real, colocando em prática os conceitos estudados.

## Desafios Encontrados

Durante o desenvolvimento deste projeto enfrentei alguns desafios importantes:

### 1. Estrutura de menus com while

No início tive dificuldade para entender como manter o sistema em execução sem que ele encerrasse após uma única operação.

Aprendi a utilizar:

while Tue e breake

para controlar a execução do programa.

## 2. Entender listas e dicionários

Um dos maiores desafios foi compreender a diferença entre:

[] (Listas) e {} (Dicionários)

Aprendi que a lista armazena vários alunos e que cada aluno pode ser representado por um dicionário contendo suas informações.

## 3. Acessar informações do dicionário

Enfrentei dificuldades ao acessar dados específicos.

Por exemplo:

aluno['valor']

Aprendi que as chaves precisam ser acessadas corretamente utilizando aspas.

## 4. Problemas de indentação

Aprendi que a indentação é uma parte fundamental do Python e que um simples desalinhamento pode gerar erros na execução do programa.

## 5. Trabalhar com condicionais

Pratiquei bastante o uso de:
if
elif
else
para validar opções do menu e a escolha dos planos.
